# BANANA

바나나 인터랙션 사이

## 📌 프로젝트 개요

바나나의 신선함과 후숙 과정에 따른 변화를 사용자의 스크롤 흐름에 맞춘 인터랙티브 요소로 시각화했습니다. 메인 섹션에 비대칭 타이포그래피와 레이어 겹침 기법을 적용하여 오브젝트가 글자 사이를 관통하는 듯한 입체적인 공간감을 구현했습니다. Bounce Ease 함수로 바나나가 낙하하는 생동감 있는 인터랙션을 설계했습니다. [텍스트-이미지-텍스트]로 이어지는 순차적 애니메이션 구성을 통해 시각적 몰입도를 높였습니다.

이미지와 텍스트의 등장 시점에 미세한 시차를 주어 정보가 순차적으로 인지되게 했습니다.

## 🛠 사용 기술

* HTML5
* CSS3 
* JavaScript 

## ✨ 주요 기능

* 입체적 레이아웃
* 반응형 웹사이트 제작
* 메인 섹션 애니메이션
